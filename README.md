# Testris game

Simple tetris game using canvas and some good old-fasioned OOP.

## How to start

Clone the repository and run:

```
yarn
yarn start
```

## How to play

- `left` - move left
- `right` - move right
- `up` - rotate
- `down` - move down
- `space` - drop immediately
- `q` - rotate left
- `e` - rotate right (same as `up`)
- `w` - drop immediately
- `p` - pause
